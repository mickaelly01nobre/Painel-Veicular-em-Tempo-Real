# Real-Time Vehicle Monitoring System (ESP32 + Ultrasonic + IMU)

## 📌 Overview
This project presents the development of a **real-time vehicle monitoring system** based on the **ESP32 microcontroller**, integrating distance sensing and motion analysis.

The system combines an **ultrasonic sensor** for obstacle detection and an **IMU (gyroscope + accelerometer)** for motion tracking, with data visualization through a **web interface**.

It is designed to improve **driving safety**, assist in **parking**, and provide **real-time vehicle behavior monitoring**.

---

## ⚙️ Features
- 🚗 **Real-Time Monitoring**
  - Continuous data acquisition and processing
  - Live updates via web interface

- 📏 **Obstacle Detection**
  - Ultrasonic sensor for distance measurement
  - Collision avoidance support
  - Parking assistance

- 📊 **Motion Analysis**
  - Gyroscope + accelerometer (IMU)
  - Detects:
    - Acceleration and deceleration
    - Direction (left/right movement)
    - Vehicle state (stopped or moving)

- 🌐 **Web Interface**
  - Real-time data visualization
  - User-friendly monitoring dashboard

- 📡 **Wireless Connectivity**
  - ESP32 with Wi-Fi communication

---

## 🧠 System Architecture
- **Microcontroller:** ESP32  
- **Sensors:**
  - Ultrasonic Sensor (distance measurement)  
  - IMU (Gyroscope + Accelerometer)  

- **Communication:**
  - Wi-Fi (ESP32)

- **Interface:**
  - Web-based dashboard

---

## ⚡ System Operation
The system operates in real time as follows:

1. The **ultrasonic sensor** measures the distance between the vehicle and nearby objects.
2. The **IMU sensor** captures motion data:
   - Acceleration
   - Direction changes (left/right)
   - Movement state (stopped or in motion)
3. The **ESP32 processes all data** and sends it via Wi-Fi.
4. The data is displayed on a **web interface**, allowing real-time monitoring.
