# Sistema de Monitoramento Veicular em Tempo Real (ESP32 + Ultrassônico + IMU)

## 📌 Visão Geral
Este projeto apresenta o desenvolvimento de um **sistema de monitoramento veicular em tempo real** baseado no **microcontrolador ESP32**, integrando sensoriamento de distância e análise de movimento.

O sistema combina um **sensor ultrassônico** para detecção de obstáculos e uma **IMU (giroscópio + acelerômetro)** para rastreamento de movimento, com visualização dos dados através de uma **interface web**.

Ele foi projetado para melhorar a **segurança na condução**, auxiliar no **estacionamento** e fornecer **monitoramento em tempo real do comportamento do veículo**.

---

## ⚙️ Funcionalidades
- 🚗 **Monitoramento em Tempo Real**
  - Aquisição e processamento contínuo de dados
  - Atualizações em tempo real via interface web

- 📏 **Detecção de Obstáculos**
  - Sensor ultrassônico para medição de distância
  - Suporte à prevenção de colisões
  - Assistência para estacionamento

- 📊 **Análise de Movimento**
  - Giroscópio + acelerômetro (IMU)
  - Detecta:
    - Aceleração e desaceleração
    - Direção (movimento esquerda/direita)
    - Estado do veículo (parado ou em movimento)

- 🌐 **Interface Web**
  - Visualização de dados em tempo real
  - Dashboard de monitoramento amigável

- 📡 **Conectividade Sem Fio**
  - ESP32 com comunicação Wi-Fi

---

## 🧠 Arquitetura do Sistema
- **Microcontrolador:** ESP32  
- **Sensores:**
  - Sensor Ultrassônico (medição de distância)  
  - IMU (Giroscópio + Acelerômetro)  

- **Comunicação:**
  - Wi-Fi (ESP32)

- **Interface:**
  - Dashboard baseado em web

---

## ⚡ Funcionamento do Sistema
O sistema opera em tempo real da seguinte forma:

1. O **sensor ultrassônico** mede a distância entre o veículo e objetos próximos.  
2. O **sensor IMU** captura dados de movimento:
   - Aceleração  
   - Mudanças de direção (esquerda/direita)  
   - Estado de movimento (parado ou em deslocamento)  
3. O **ESP32 processa todos os dados** e os envia via Wi-Fi.  
4. Os dados são exibidos em uma **interface web**, permitindo o monitoramento em tempo real.

# Real-Time Vehicle Monitoring System (ESP32 + Ultrasonic + IMU)

## 📌 Overview
This project presents the development of a **real-time vehicle monitoring system** based on the **ESP32 microcontroller**, integrating distance sensing and motion analysis.

The system combines an **ultrasonic sensor** for obstacle detection and an **IMU (gyroscope + accelerometer)** for motion tracking, with data visualization through a **web interface**.

It is designed to improve **driving safety**, assist in **parking**, and provide **real-time vehicle behavior monitoring**.

---

## ⚙️ Features
- 🚗 **Real-Time Monitoring**
  - Continuous data acquisition and processing
  - Live updates via web interface

- 📏 **Obstacle Detection**
  - Ultrasonic sensor for distance measurement
  - Collision avoidance support
  - Parking assistance

- 📊 **Motion Analysis**
  - Gyroscope + accelerometer (IMU)
  - Detects:
    - Acceleration and deceleration
    - Direction (left/right movement)
    - Vehicle state (stopped or moving)

- 🌐 **Web Interface**
  - Real-time data visualization
  - User-friendly monitoring dashboard

- 📡 **Wireless Connectivity**
  - ESP32 with Wi-Fi communication

---

## 🧠 System Architecture
- **Microcontroller:** ESP32  
- **Sensors:**
  - Ultrasonic Sensor (distance measurement)  
  - IMU (Gyroscope + Accelerometer)  

- **Communication:**
  - Wi-Fi (ESP32)

- **Interface:**
  - Web-based dashboard

---

## ⚡ System Operation
The system operates in real time as follows:

1. The **ultrasonic sensor** measures the distance between the vehicle and nearby objects.
2. The **IMU sensor** captures motion data:
   - Acceleration
   - Direction changes (left/right)
   - Movement state (stopped or in motion)
3. The **ESP32 processes all data** and sends it via Wi-Fi.
4. The data is displayed on a **web interface**, allowing real-time monitoring.
